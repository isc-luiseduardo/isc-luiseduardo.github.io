Proyectos Tec Uruapan

Durante mi estancia como estudiante, he desarrollado algunas aplicaciones para el Instituto Tecnológico Superior de Uruapan. Entre otros proyectos menores realizados, cuento en mi haber con los siguientes:

[Repositorio Kiosko] - 2015 Fue una solución web desarrollada en ASP.NET (mi primer proyecto real en este Framework) con el propósito de compartir información digital en PDF acerca de las distintas carreras, con el propósito de tener un acervo de información confiable y validada para la realización de tareas e investigaciones.

[Website Cyber-GT] - 2016 Un sitio web realizado para promocionar y conectar el negocio de mi familia (Un cyber y servicio de artículos promocionales por sublimación, así como el Taller Loto), con el propósito de realizar el protocolo de investigación para la materia 'Taller de Investigación I y II'. Está realizado en ASP.NET con arquitectura MVC 5.

[Bitácora móvil LogExpress] - 2016 Aplicación móvil desarrollada en Cordova con el Framework Ionic, con el propósito de llevar un registro de información correspondiente a las instalaciones de redes de computadoras. Usa API's para control de la cámara, almacenamiento interno mediante SQLite, GPS y gestor de archivos.

[Farmacia Magaña] - 2016 Aplicación web desarrollada como punto de venta y control para la Farmacia Magaña, ubicada en la población de Corupo, Michoacán, para llevar controles de inventario básicos. Está desarrollada en ASP.NET, y esta aplicación ya fue desarrollada más apegada a las buenas prácticas de desarrollo. Está desarrollada bajo el paradigma MVC y usa la arquitectura de tres capas (Interfaz de usuario, capa de lógica de negocios y capa de acceso a datos).

[Juegos] - 2012 - 2013 Se desarrollaron un par de juegos en Java para las materias básicas de programación. Uno de ellos era un juego tipo Night Raid (juego de DOS) y el otro era un juego tipo Snake.
  • Periodo comprendido: 2012 - 2016
  • Lenguajes: Java, C#, javascript
  • Áreas: Aplicaciones web, aplicaciones móviles, juegos, programas de escritorio.
