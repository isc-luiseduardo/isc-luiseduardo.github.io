Yuggoth Soft

Yuggoth Soft es mi segunda empresa, que nace a raíz de mi primer desarrollo de Software freelance, el cual fue un sistema de control para la Veterinaria 'Don Balta' (Ziracuaretiro), realizado en Java y conectado a MySQL en forma local. Además, con mi hermano comenzamos a desarrollar una app móvil en nuestros ratos libres y que actualmente sigue en desarrollo. En Playstore sólo está una aplicación que desarrollé para la escuela (Bitácora), la cual fue desarrollada en Cordova y usando el Framework Ionic. La aplicación que desarrollamos es una herramienta para la gestión de bandas musicales y actualmente se encuentra en su fase inicial, se desarrolla en Xamarin.Forms y se implementará con API's en Python y una base de datos en SQL Server.
